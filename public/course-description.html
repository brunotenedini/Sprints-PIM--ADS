Sobre o Curso

O curso de Fundamentos HTML é ideal para quem deseja iniciar sua jornada no desenvolvimento web. Aprenda a estruturar páginas da web utilizando a linguagem HTML, a base de todas as páginas da internet.

O que você vai aprender:

  • Estruturas básicas de HTML
  • Tags HTML essenciais
  • Formatação de texto
  • Links e imagens
  • Semântica em HTML
